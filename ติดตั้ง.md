# วิธีติดตั้งโปรแกรม (ฉบับเข้าใจง่าย)

โปรแกรมนี้ไว้ **ตัดต่อเสียงพูดภาษาไทยด้วยการแก้ตัวหนังสือ** — พิมพ์แก้คำ กดลบคำที่ไม่เอา แล้วเสียงก็ตัดตาม

---

## 📥 ขั้นที่ 1 — โหลดโปรแกรม

1. เข้าหน้านี้: **[หน้าดาวน์โหลด (Releases)](../../releases/latest)**
2. เลื่อนลงไปหาหัวข้อ **Assets** (แปลว่า "ไฟล์")
3. กดโหลดไฟล์ให้ตรงกับเครื่องของคุณ:
   - ใช้ **Windows** → โหลดไฟล์ที่ลงท้ายด้วย **`.exe`**
   - ใช้ **Mac** → โหลดไฟล์ที่ลงท้ายด้วย **`.dmg`**

> ยังไม่มีไฟล์ให้โหลด? แปลว่ายังไม่ได้ออกเวอร์ชัน — บอกคนที่ดูแลโปรเจกต์ให้ "ออก release" ให้

---

## 💻 ขั้นที่ 2 — เปิดโปรแกรมครั้งแรก

โปรแกรมนี้ทำเอง ไม่ได้เสียเงินซื้อใบรับรองจาก Microsoft/Apple เครื่องเลย **เตือนว่าไม่รู้จัก** เป็นเรื่องปกติ ทำแบบนี้ครั้งเดียวพอ:

### ถ้าใช้ Windows
1. ดับเบิลคลิกไฟล์ `.exe`
2. ถ้าขึ้นหน้าจอฟ้า **"Windows protected your PC"** → กดคำเล็กๆ ว่า **More info**
3. แล้วกดปุ่ม **Run anyway** (แปลว่า "รันเลย")
4. ถ้ามีหน้าต่างถามเรื่อง **Firewall** → กด **Allow** (อนุญาต)

### ถ้าใช้ Mac
1. ดับเบิลคลิกไฟล์ `.dmg` → ลากรูปโปรแกรมไปใส่โฟลเดอร์ **Applications**
2. ไปที่ Applications → **คลิกขวา** ที่โปรแกรม → เลือก **Open**
3. มีหน้าต่างเตือน → กด **Open** อีกที (ทำแบบนี้แค่ครั้งแรกครั้งเดียว)

## 🎯 ขั้นที่ 3 — เลือกว่าเครื่องนี้จะทำอะไร

เปิดโปรแกรมครั้งแรกจะเห็น **แถบสีเหลืองด้านบน** เขียนว่า "ยังไม่มีโมเดล AI"
คุณจะ **กด หรือ ไม่กด** ก็ได้ แล้วแต่ว่าเครื่องนี้ทำหน้าที่อะไร:

### 🅰️ ถ้าเครื่องนี้ต้อง "ถอดเสียงเป็นตัวหนังสือ" เอง

> เหมาะกับ: เครื่องหลักที่รับไฟล์เสียงมาแล้วให้ AI ถอดเป็นข้อความ

1. กดปุ่ม **"ติดตั้งโมเดล"** ในแถบเหลือง
2. รอโหลด (ประมาณ 4-5 GB ใช้เวลาสักพัก **โหลดครั้งเดียวจบ** ต้องมีเน็ตตอนนี้)
3. โหลดเสร็จ แถบเหลืองหายไปเอง → ใช้ปุ่ม **"ถอดเสียง"** กับ **"ตรึงบท"** ได้
4. หลังจากนี้ **ไม่ต้องต่อเน็ตอีกเลย** ใช้งานในเครื่องได้หมด

*(เครื่องนี้ควรมีแรม 16GB ขึ้นไป และพื้นที่ว่างสัก 6GB)*

### 🅱️ ถ้าเครื่องนี้แค่ "เอางานที่คนอื่นถอดมาแล้ว มาตัด/ตรวจ"

> เหมาะกับ: เครื่องของคนตัดต่อ หรือคนตรวจงาน — เครื่องธรรมดาก็พอ

1. **ไม่ต้องกด "ติดตั้งโมเดล"** — ปล่อยแถบเหลืองไว้เฉยๆ (ไม่กวนการทำงาน)
2. กดปุ่ม **"เปิดไฟล์เสียง"** → เลือกไฟล์งานที่เพื่อนส่งมา (ไฟล์ลงท้าย **`.audioedit.json`**)
3. ทำได้เลยทันที: ฟังเสียง, แก้คำ, ลบคำที่ไม่เอา, **ตัดเสียง**, และ **Export** (บันทึกเสียงที่ตัดเสร็จ)

*(มีแค่ปุ่ม "ถอดเสียง"/"ตรึงบท" ที่กดไม่ได้ นอกนั้นใช้ได้หมด — เครื่องแรม 8GB ก็พอ)*

## 📀 ก๊อปโมเดลไปเครื่องที่ต่อเน็ตไม่ได้ (แจกโมเดลด้วย USB)

ถ้าเครื่องอื่น **ต่อเน็ตไม่ได้** ก็ไม่ต้องกด "ติดตั้งโมเดล" (ที่ต้องโหลดจากเน็ต) —
ให้ **ก๊อปโฟลเดอร์โมเดลจากเครื่องที่โหลดไว้แล้ว** ใส่ USB ไปวางแทน โมเดลเป็นแค่ไฟล์
ก๊อปได้ปกติ ใช้งานได้เหมือนกันเป๊ะ

**ที่เครื่องที่มีโมเดลแล้ว** — หาโฟลเดอร์ชื่อ `models` ก๊อปใส่ USB:
- **Windows**: เปิด File Explorer → พิมพ์ตรงช่องที่อยู่ด้านบนว่า
  `%APPDATA%\com.thaiaudioeditor.app` แล้วกด Enter → จะเจอโฟลเดอร์ `models`
- **Mac**: เปิด Finder → เมนู **Go → Go to Folder** → วาง
  `~/Library/Application Support/com.thaiaudioeditor.app` → จะเจอโฟลเดอร์ `models`

> โฟลเดอร์ `models` ใหญ่ประมาณ **4.5 GB** (ถ้าเห็นโฟลเดอร์ `pathumma-large-v3` ด้วย
> อันนั้นเป็นตัวเสริม ~3GB ไม่ก๊อปก็ได้ ประหยัดที่)

**ที่เครื่องปลายทาง (ต่อเน็ตไม่ได้)**:
1. ติดตั้งโปรแกรมก่อน (เอาไฟล์ `.exe`/`.dmg` ใส่ USB ไปติดตั้งตามขั้นที่ 1-2 ข้างบน)
2. **เปิดโปรแกรม 1 ครั้งแล้วปิด** (เพื่อให้มันสร้างโฟลเดอร์เก็บข้อมูลขึ้นมา)
3. ไปที่โฟลเดอร์เดียวกัน (ใช้วิธีข้างบน `%APPDATA%\...` / `~/Library/...`)
4. **วางโฟลเดอร์ `models` ที่ก๊อปมา** ลงตรงนั้น
5. เปิดโปรแกรมอีกที → แถบเหลืองหายไปเอง → ใช้ถอดเสียง/ตรึงบทได้เลย ไม่ต้องต่อเน็ต

## 📨 การส่งงานให้กัน

- **ส่งให้คนอื่นทำต่อ:** เอาไฟล์เสียงต้นฉบับ + ไฟล์ `.audioedit.json` **ใส่โฟลเดอร์เดียวกัน** แล้วบีบ (zip) ส่งไป
  → ปลายทางเปิดไฟล์ `.audioedit.json` ในโปรแกรมได้เลย เห็นงานครบทุกอย่าง
- **ส่งงานที่เสร็จแล้ว:** กดปุ่ม **Export** → ได้ไฟล์เสียง `.wav` ที่ตัดเรียบร้อย + ไฟล์ข้อความ `.docx`/`.txt`

## ❓ เจอปัญหา?

- **เปิดโปรแกรมแล้วปุ่มถอดเสียงกดไม่ได้ (สีเทา)** → ยังไม่ได้ติดตั้งโมเดล (ดูขั้นที่ 3 แบบ 🅰️)
- **กดเปิดไฟล์ .audioedit.json แล้วหาไฟล์เสียงไม่เจอ** → เอาไฟล์เสียงต้นฉบับมาไว้โฟลเดอร์เดียวกับไฟล์ .audioedit.json
- **ปัญหาอื่นๆ** → ถ่ายรูปหน้าจอส่งให้คนดูแลโปรเจกต์
