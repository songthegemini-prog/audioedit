# คู่มือการใช้ + ปุ่มลัดทั้งหมด

> วิธีติดตั้งโปรแกรมดูที่ [ติดตั้ง.md](ติดตั้ง.md) — หน้านี้คือวิธี "ใช้งาน"

## ขั้นตอนการทำงาน (ภาพรวม)

1. **เปิดไฟล์เสียง** (หรือไฟล์งาน `.audioedit.json` ที่เพื่อนส่งมา — ใช้ปุ่มเดียวกัน)
2. ได้ตัวหนังสือมา 2 ทาง: **ถอดเสียง** (ให้ AI ถอด) หรือ **ตรึงบท** (มีบท
   .txt/.docx อยู่แล้ว — แม่นกว่า และถ้ามีคนแก้บทใน Word มาให้ ใช้ทางนี้เลย)
3. แก้บท 3 แบบ (สามอย่างนี้**ต่างกัน**):
   - **แก้คำสะกด** = ดับเบิลคลิกที่คำ → พิมพ์ → Enter (เสียงไม่ถูกแตะ)
   - **ซ่อนคำจากบท** (อ่า/อืม/เสียงไอ) = คลิกขวาที่คำ (เสียงยังอยู่ แค่ไม่ไปอยู่ในเอกสาร)
   - **ตัดเสียง** = เลือกคำ → Delete (เสียง+ตัวหนังสือหายพร้อมกัน ย้อนได้ด้วย Undo)
4. **บันทึก** (Ctrl/Cmd+S) ได้ไฟล์งาน `.audioedit.json` — ส่งต่อให้เพื่อนได้
   (zip คู่กับไฟล์เสียงต้นฉบับในโฟลเดอร์เดียวกัน)
5. **Export** → เลือก "เสียง + บท" หรือ "เฉพาะบท" — ไฟล์ต้นฉบับไม่ถูกแก้เลยทุกกรณี

## เคล็ดการตัดให้คม

- ดูบน **spectrogram** (แถบล่าง): **แถบมืดแนวตั้ง = ช่วงเงียบระหว่างคำ** —
  ตัดตรงนั้นปลอดภัยสุด
- ระวัง**ฝ้าจางๆ ด้านบน**หลังคำที่ลงท้าย ส/ฟ/ช — คือหางเสียงที่ waveform
  มองไม่เห็น ตัดโดนแล้วคำจะขาดห้วน
- ปกติโปรแกรม snap ขอบเข้าช่วงเงียบให้อัตโนมัติ + ใส่ crossfade กันเสียงคลิก
  ทุกรอยต่ออยู่แล้ว — ปุ่ม **±1ms/±10ms** ใช้ตอนฟังแล้วยังไม่เนียน:
  ขยับทีละนิด → Shift+Space ฟังซ้ำ → **หูคือเกณฑ์ตัดสินสุดท้าย**

## ⌨️ ปุ่มลัดคีย์บอร์ด

| ปุ่ม | ทำอะไร |
|---|---|
| **Space** | เล่น / หยุด |
| **Shift+Space** | ฟังเฉพาะช่วงที่เลือก |
| **Tab / Shift+Tab** | กระโดดไปคำขีดแดง (ที่ระบบไม่มั่นใจ) ถัดไป/ก่อนหน้า + เล่นเสียง + เปิดกล่องแก้ |
| **Enter** | เปิดกล่องแก้คำที่เลือก |
| **Delete / Backspace** | ตัดช่วงที่เลือก |
| **Escape** | ยกเลิกการเลือก |
| **Ctrl/Cmd+S** | บันทึกโปรเจกต์ |
| **Ctrl/Cmd+F** | ค้นหาคำ |
| **Ctrl/Cmd+Z** | ย้อนการตัด (Undo) |
| **Ctrl/Cmd+Shift+Z** | ทำซ้ำ (Redo) |
| **Ctrl/Cmd+ล้อเมาส์** (บนคลื่นเสียง) | ซูมเข้า/ออก |

## 🖱️ เมาส์บนตัวหนังสือ

| ท่า | ทำอะไร |
|---|---|
| คลิกคำ | เลือกคำ (เสียงเด้งตาม ไม่เล่นเอง — จะฟังกด Space) |
| Shift+คลิก | เลือกตั้งแต่คำที่เลือกไว้ถึงคำนี้ |
| ลากกวาด | เลือกหลายคำ |
| ดับเบิลคลิก | แก้คำสะกด (เสียงไม่ถูกแตะ) |
| คลิกขวา | ซ่อนคำจากบท (เสียงยังอยู่) |
| ปุ่ม ✎ หน้าวรรค | แก้ทั้งวรรคอิสระ แล้วระบบตรึงเวลาใหม่ให้ |

## 🖱️ เมาส์บนคลื่นเสียง (waveform + spectrogram)

| ท่า | ทำอะไร |
|---|---|
| คลิก | ย้ายหัวอ่านไปตรงนั้น |
| ลากบนที่ว่าง | เลือกช่วงเสียง (ไม่ต้องมีบทก็ตัดได้ — ตัดหยาบก่อนถอด) |
| ลากขอบแถบฟ้า/แดง | ปรับขอบช่วงเลือก/ขอบตัด (ทำได้ทั้งสองมุมมอง) |
| ปุ่ม ±1ms ±10ms (แถบล่าง) | ขยับขอบตัดละเอียดทีละนิด |

## 🎨 ความหมายสี/สัญลักษณ์

| เห็น | คือ |
|---|---|
| คำขีดเส้นแดง | ระบบไม่มั่นใจ ควรตรวจ (กด Tab ไล่ได้) |
| คำจาง+ขีดทิ้ง | ถูกซ่อนจากบท (เสียงยังอยู่) |
| แถบฟ้าบนคลื่น | ช่วงที่เลือก (ยังไม่ตัด) |
| แถบแดงบนคลื่น | ช่วงที่ตัดแล้ว (ย้อนได้ด้วย Ctrl/Cmd+Z) |
| ● ยังไม่บันทึก | มีงานค้าง อย่าลืม Ctrl/Cmd+S |
| จุดเขียว "backend: connected" (มุมล่าง) | ตัวประมวลผลพร้อมทำงาน — ถ้าแดง ปิดโปรแกรมแล้วเปิดใหม่ |
